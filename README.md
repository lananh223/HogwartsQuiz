# HogwartsQuiz

**Hogwarts Quiz** is a native Android application that features questions related to Harry Potter and Hogwarts in a quiz format. It's a simple app that I developed while learning the basics of Android.

  * Activity
  * Explicit Intents
  * MVC Design
  * SharedPreference
  * View Binding
